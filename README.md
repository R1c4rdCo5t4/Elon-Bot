# Elon Bot
Simple Discord bot with slash commands made with discord.py


### Elon can:
> Say hello/goodbye

> Play Rock Paper Scissors

> Be your calculator

> Change Status

> Flip a coin

> Add a role to an user

> Show user information

> Translate a sentence to the selected language

> Send a random meme

> Send a private message
